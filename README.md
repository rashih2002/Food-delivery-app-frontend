# 🍔 FoodGo – Food Delivery App (Frontend Prototype)

## 📖 Overview
FoodGo is a food delivery web application prototype developed using HTML, CSS, and JavaScript. It provides a simple and user-friendly interface for browsing food items, adding them to cart, and placing orders.

This project focuses only on frontend development and UI design.

---

## 🛠 Technologies Used
- HTML5  
- CSS3  
- JavaScript  

---

## ✨ Features
- Home / Landing page  
- User login & registration UI  
- Food item listing  
- Product details page  
- Shopping cart  
- Checkout page  
- Order tracking UI  
- User profile page  
- Customer support page  
- Responsive design
